(Ouvrez-moi dans un navigateur) Exercices de JS
Consigne : Pour chaque exo, il faut ouvrir le dossier correspondant, où vous trouverez un fichier js que vous pouvez modifier.
Les consignes sont marquées sur le haut du fichier js et il ne faut surtout pas les effacer.
Pour lire une ligne d'entrée (input() en python), vous pouvez faire readline() directement (la fonction n'existe pas nativement sur le js du web mais sera prise en charge)
Pour afficher une ligne, faites (console.log('texte')). Regardez d'abord l'exo-exemple, ça vous sera utile.
Les exos sont classés par ordre de difficulté.
Vous pouvez utiliser DevTools (Ctrl + Shift + I) pour voir les erreurs/logs
